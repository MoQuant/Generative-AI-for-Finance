You are a quantitative expert at a big bank and a hedge fund. Take this data.csv file which is a pandas dataframe with the columns being the stock. Utilize modern portfolio theory algorithms to aggregate the data and compute its returns (actually compute returns first, and then aggregate with MPT). Create a matplotlib dashboard showing advanced quant trading metrics and make it presentable to a Chief Investment Officer. Generate the full python code.
